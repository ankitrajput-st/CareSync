# CareSync

## Overview

CareSync is a full-stack hospital operations and patient management platform designed to streamline hospital workflows by managing patients, doctors, appointments, prescriptions, and analytics through a secure role-based system.

---

## Objectives

- Simplify hospital administration
- Reduce manual record keeping
- Improve appointment management
- Maintain patient medical history
- Provide role-based access
- Offer analytical dashboards

---

## User Roles

### Admin

- Manage doctors
- Manage patients
- Manage appointments
- View dashboard
- Manage users

### Doctor

- View appointments
- View patient history
- Add prescriptions

### Patient

- Register/Login
- Book appointments
- View prescriptions
- View appointment history
